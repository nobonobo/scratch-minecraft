# scratch-minecraft

- for Minecraft Java Edition 1.16.5
- Forge +「Remote Control from Scratch」

## prerequire for build

```
make setup
```

## build

```
make build
```

## run

```shell:Windows
./build/bin/scratch-minecraft.exe
```
```shell:macOS
open ./build/bin/scratch-minecraft.app
```
```shell:Linux
./build/bin/scratch-minecraft
```
